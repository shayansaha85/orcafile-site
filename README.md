# 🐋 OrcaFile

> Fast File Organizer & Visualizer

A fast, lightweight desktop application that scans any folder or drive and **instantly groups every file by its extension**.

Built with a premium Nord-inspired interface and a blazing-fast multi-threaded indexing engine for navigating massive directories effortlessly.

# ✨ Features

## ⚡ Blazing Fast Indexing

A custom two-pass background indexing engine scans massive local and external drives without freezing the UI, with accurate progress tracking.

---

## 🎨 Premium Nord Theme

A carefully crafted, high-contrast workspace featuring rounded corners, subtle hover effects, and streamlined controls for extended usage.

---

## 🔍 Accumulative Search

Search intelligently without losing selections.

Search for an extension, select it, clear the search, and continue filtering while preserving previous selections.

---

## 🌲 Deep File Tree Filtering

Quickly locate files within the indexed directory tree.

Non-matching groups are automatically hidden to reduce visual clutter.

---

## 📂 Native Explorer Integration

Double-click any file or folder to instantly reveal its location in Windows File Explorer.

---

## 🚀 Multi-Threaded Engine

Background indexing never blocks the UI, ensuring smooth performance even while scanning very large drives.

# 📦 Installation

### Step 1

Download:

`OrcaFile_Setup.exe`

from the Releases page.

---

### Step 2

Run the installer.

The setup wizard automatically:

- installs all required dependencies
- installs OrcaFile into **Program Files**

---

### Step 3

Launch OrcaFile from:

- Desktop
- Start Menu

and start organizing your files.
